# Titanic Survival Prediction

This project is a Machine Learning based Titanic Survival Prediction system that predicts whether a passenger survived or not based on different passenger details.

The model is built using Logistic Regression and deployed using Streamlit for interactive predictions.

---

## Features

- Titanic survival prediction system
- Data preprocessing and feature engineering
- Categorical data encoding using LabelEncoder
- Feature scaling using StandardScaler
- Logistic Regression classification model
- Model saved using Pickle (`model.pkl`)
- Interactive Streamlit web application
